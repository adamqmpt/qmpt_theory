# QMPT Lab IDE – Overview

Version: 0.3.0 (planned features baked into 0.2.x baseline + run history/log viewer).

## What this tool does (EN)

- Dark-theme Tkinter IDE for QMPT documents and experiments.
- Browse and view Markdown theory files.
- Edit/save lab notes with live Markdown/inline-math preview.
- Insert experiment templates with auto-filled config/seed/device/timestamp/log.
- Launch placeholder simulations from configs; logs are written under `lab/logs/`.
- Run history with statuses and in-app log viewer.

How to run:

```
python3 -m code.qmpt_ide.app
```

## Что делает этот инструмент (RU)

- Тёмная IDE на Tkinter для файлов QMPT и экспериментов.
- Просмотр Markdown-документов теории.
- Редактирование и сохранение заметок с превью Markdown/инлайн-формул.
- Вставка шаблонов эксперимента с авто-заполнением config/seed/device/timestamp/log.
- Запуск заглушки симуляций по конфигам; логи пишутся в `lab/logs/`.
- История прогонов со статусами и просмотр логов внутри IDE.

Запуск:

```
python3 -m code.qmpt_ide.app
```
